# DayZ Battle Royale

Open source DayZ Battle Royale mod code and tools.

## Developer Notes

I had these files laying around and didn't want them to go to waste. So here
they are. Feel free to explore them, but they are not officially supported or
documented. I don't want to deal with support requests for them.

- [client](./client/) - DayZBR Client API
- [server](./server/) - DayZBR Server API
- [leaderboard](./leaderboard/) - DayZBR Leaderboard API
- [mod](./mod/) - DayZ Mod Code & Tools

All other files are in support of the above main services and projects.
